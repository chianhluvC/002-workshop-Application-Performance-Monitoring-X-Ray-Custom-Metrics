Tạo Lambda Function có Tracing

Trong bước này bạn sẽ tiến hành tạo một ứng dụng serverless có tracing, để theo dõi quá trình khi người dùng thực hiện truy vấn tới ứng dụng đó.

  1. Truy cập vào giao diện quản trị của AWS Lambda Console và click chọn vào Create a funtion

  1. Giao diện create function xuất hiện
  • Click chọn Author from scratch
  • Funtion name điền MonitorFuction
  • Runtime click chọn vào phiên bản Node.js mới nhất
  • Architecture click chọn vào x86_64

  1. Kéo xuống thêm một đoạn
  • Click chọn vào Create a new role with basic Lambda permission
  • Click vào Create function

  1. Sau khi hoàn tất thao tác trước đó, function vừa tạo sẽ xuất hiện

  • Click chọn vào mục Code:
  • Paste đoạn code sau vào và ấn deploy

  export async function handler(event) {
      const start = Date.now();
	
  	  await new Promise((r) => setTimeout(r, Math.random() * 300 + 300));
  	  const latency = Date.now() - start;
	
  	  console.log("Request processed in:", latency, "ms");
	
  	  console.log(JSON.stringify({
  	    "_aws": {
  	      "Timestamp": Date.now(),
  	      "CloudWatchMetrics": [
  	        {
            "Namespace": "MyApp",
	          "Dimensions": [["Service"]],
	          "Metrics": [{ "Name": "ProcessingTime", "Unit": "Milliseconds" }]
	        }
	      ]
	    },
	    "Service": "CheckoutService",
	    "ProcessingTime": latency
	  }));
	
	  return {
	    statusCode: 200,
	    body: JSON.stringify({
	      message: "Hello! Tracing + metrics working.",
	      latency
	    })
	  };	
  }

  1. Click chọn vào mục Configuration và chọn vào Monitoring and operations tools

  1. Trong mục Monitoring and operations tools click chọn vào Edit

  1. Giao diện Edit monitoring tools sẽ xuất hiện, phần Lambda service traces click chọn vào Enable

  1. Kéo xuống cuối trang và click chọn vào Save

  1. Xuất hiện thông báo đã thay đổi thành công
  • Quan sát Logs and metrics (default) hiện Enabled
  • Quan sát Lambda service traces hiện Enabled

  1. Quay lại MonitorFuntion và click chọn vào Add trigger

  1. Giao diện add triggers sẽ xuất hiện
  • Phần Intent click chọn vào Create a new API
  • Phần API type click chọn vào HTTP API
  • Phần Security click chọn vào Open
  • Click chọn vào Add

  1. Xuất hiện thông báo và trigger đã được tạo thành công
